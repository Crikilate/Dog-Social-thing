How to Use the Dog Social Website

The "Dog Social" button (first button) is to return back to the main menu.

The "YouTube" button is to view videos of dogs.

The "How to Use" button is to view this page.

The "More Info" button is to view information about dog breeds which you will select.

The "Contact" button is to contact the developers of this website.

The "Log out" button is to log out of your account.
